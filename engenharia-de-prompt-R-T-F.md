# Engenharia de Prompt — Framework R-T-F (Role-Task-Format)

## Quando utilizar o Framework R-T-F

O R-T-F é ideal para tarefas de domínio público e conhecimento padronizado (como Dockerfiles, IAM Policies e scripts), onde a IA já possui base técnica consolidada.

Não o utilize em cenários que exigem raciocínio crítico, análise de logs específicos (como CrashLoopBackOff) ou decisões estratégicas (migrações e arquitetura), pois estes demandam contextos e variáveis de negócio que o framework simplificado não abrange.
