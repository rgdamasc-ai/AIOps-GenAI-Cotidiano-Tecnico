# Engenharia de Prompt — Framework B-A-B (Before-After-Bridge)

## Quando utilizar o Framework B-A-B

O Framework B-A-B (Before-After-Bridge) deve ser utilizado em contextos de transformação, onde você define um estado atual problemático e um estado futuro desejado para que a IA trace o caminho técnico. Não deve ser usado para tarefas diretas ou criações do zero (como resumos ou dúvidas conceituais), onde frameworks de contexto simples são mais eficazes.

## Estrutura do Prompt

```
# Before
[Espaço para texto]

# After
[Espaço para texto]

# Bridge
[Espaço para texto]
```
